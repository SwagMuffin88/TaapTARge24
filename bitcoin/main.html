Bitcoin kalkulaator

Tegemist on lihtsa Bitcoin kalkulaatoriga, mis võtab kasutajalt sisendina soovitud teisendatava lähtesumma ning sihtvaluuta nimetuse lühendina, seejärel sooritab teisenduse Coinbase API abil bitcoin hetkeväärtusest lähtudes.

Kui kasutaja jätab Bitcoin väärtuse sisestamata või sisestab midagi muud peale arvväärtuse, kuvatakse hüpikaken veateatega ning sisendit küsitakse seni, kuni sisestatakse korrektne lähtearvväärtus. Kui kasutaja jätab sisestamata soovitud sihtvaluuta või sisestab ebakorrektse sõne, kuvatakse samuti hüpikaken veateatega ning sisendit küsitakse uuesti.
