Lower CASE vahendid

Definitsioon

Teostusekesksed, mudelitest reaalse tarkvaratoote loomise käigus kasutatavad vahendid. Toetavad andmebaasi struktuuri genereerimist, koodi genereerimist, testide läbiviimist, koodi versioonihaldust, konfiguratsioonihaldust, pöördprojekteerimist jms. Lower CASE vahendid on enamasti seotud "back end" osaga tarkvaraarendusest ning levinumad tööriistad on koodigenereerijad, kompileerijad ja testimist hõlbustavad tööriistad.


Isiklik kogemus

Kuna olen juba paar aastat tegelenud Java arendusega, olen paljude Lower CASE tööriistade esindajatega kokku puutunud. Raske on öelda täpselt, millistega ma pole kokku puutunud, sest ma ei tea kõiki. Olen peamiselt kokku puutunud IDE-dest IntelliJ, Visual Studio Code ja Visual Studioga. Kasutan ka Git version controli. Arenduse käigus testimisel kasutan palju ka debuggin tööriistu IntelliJ-siseselt.
